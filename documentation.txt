# Golang Properties
    When the variable is declared and receives a value at the same time, the 
    compiler is smart enough to guess the variable type.

    Go only has one loop structure, which is for.

    Declaring a variable and not using it may cause a compilation error.
    Due to this property, you may use the blank identifier (_).
